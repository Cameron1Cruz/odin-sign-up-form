# Sign-Up Form Project

This project is part of the **Odin Project: Intermediate HTML & CSS** course.  
It demonstrates building a professional-looking sign-up form layout using **HTML, CSS, and Flexbox**.

---

## Features

### Layout
- Two-column **Flexbox layout**:  
  - Left column: **sidebar** with a background image and dark overlay  
  - Right column: **main content** containing the sign-up form

### Sidebar
- Full **background image** covering the sidebar  
- **Dark semi-transparent overlay** for readability  
- **Photo credit** positioned at the bottom

### Sign-Up Form
- Centered form container with **white card background** and **subtle box-shadow**  
- Input fields for:
  - Name
  - Email
  - Password
  - Confirm Password
- Submit button styled with **green color** matching the design tone

### Form Validation (CSS only)
- Inputs display **blue border + glow** on focus (`:focus`)  
- Inputs display **red border** when invalid (`:invalid`)  
- HTML attributes used for required fields and basic validation  
- Password matching validation **not implemented** (JavaScript required in future lessons)

### Styling & UX
- Clean and consistent **spacing and typography**  
- Rounded inputs and buttons for modern look  
- Responsive to viewport width (horizontal centering, full-height layout)

---

## Technologies Used
- HTML5  
- CSS3 (Flexbox, pseudo-classes, box-shadow, background-image)  

---
